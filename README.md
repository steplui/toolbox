# Toolbox 小工具集合

13 個日常小工具 Dashboard，支援 PWA 安裝到手機主畫面。

## 部署到 GitHub Pages

### 第一次設定

1. 登入 [github.com](https://github.com)，撳右上角 **+** → **New repository**
2. Repository name 輸入 `toolbox`
3. 設為 **Public**，然後撳 **Create repository**
4. 將呢個資料夾嘅所有檔案上傳到 repo
5. 去 repo 嘅 **Settings** → **Pages**
6. Source 揀 **Deploy from a branch**，Branch 揀 **main**，資料夾揀 **/ (root)**，撳 **Save**
7. 等 1-2 分鐘，你嘅網址就係 `https://<你的用戶名>.github.io/toolbox/`

### 加到 iPhone 主畫面

1. 用 **Safari** 打開你嘅 GitHub Pages 網址
2. 撳底部 **分享按鈕** ⬆️
3. 揀 **「加到主畫面」**
4. 撳 **「加入」**

完成！打開就係全屏 app 體驗。
